Chapter 1 Quiz

Each question starts with a short scenario and asks for the best MIS explanation or decision.

Question 1
Application / analysis
A student organization buys a new scheduling app and expects its event planning problems to disappear right away. After a month, officers realize that members still enter data inconsistently, nobody agrees on the process for approving events, and important deadlines are still missed. The president says the mistake was thinking the app alone would fix everything.

Which idea BEST explains why the new app did not solve the problem by itself?

A. An information system includes people and procedures, not just software
B. Network effects automatically improve every internal process
C. Switching costs always make new apps fail at first
D. Capital intensity is the main barrier to better scheduling
Correct answer: A
Explanation
A is correct because the five-component model says an information system includes hardware, software, data, procedures, and people. The scenario shows that the organization changed one component, software, but failed to fix the people and procedure parts of the system. B is wrong because network effects are about value increasing with more users, not about broken internal workflows. C is wrong because switching costs are not the main problem described. D is wrong because capital intensity has nothing to do with a student group’s scheduling process.
Question 2
Application / analysis
A small bakery wants to offer online ordering, curbside pickup, and automatic text updates. The owner already has computers and internet access, but she realizes the business also needs cleaner order data, clearer employee steps, and training so staff know how to handle pickup orders correctly. She asks what broader MIS concept explains this change.

Which concept BEST fits what the bakery is trying to do?

A. Vertical integration into delivery fleets
B. Digital transformation of a business process
C. Capital intensity in semiconductor design
D. Price elasticity of computing demand
Correct answer: B
Explanation
B is correct because the bakery is using digital tools to redesign how it takes orders, communicates with customers, and runs pickup operations. The slides frame digital transformation as more than just buying technology; it changes business processes and how work gets done. A is wrong because the bakery is not taking ownership of another stage in a value chain. C is unrelated to bakery operations. D is about demand rising as tech gets cheaper, which is not the main issue here.
Question 3
Application / analysis
A startup’s founder tells her team, “We already bought better laptops, so our information system is now stronger.” Her operations manager disagrees and points out that the company still has messy customer data, unclear task ownership, and no shared process for following up with leads. The manager wants the founder to see the difference between information technology and an information system.

Which explanation is MOST accurate?

A. Information technology and information systems mean exactly the same thing
B. Stronger laptops alone create a complete system even without good data or procedures
C. Information technology is only the hardware and software portion, while an information system is broader
D. Information systems matter only when a company becomes very large
Correct answer: C
Explanation
C is correct because the slides distinguish information technology from the broader information system. IT usually refers to the hardware and software, while an information system also includes people, procedures, and data. A is wrong because the chapter treats the two ideas differently. B is wrong because the scenario clearly shows missing system components. D is wrong because even small firms rely on information systems to operate effectively.
Question 4
Analysis / evaluation
A campus tutoring service stores student notes in one app, attendance in another, and billing details in a third. Tutors often waste time asking each other for updates because information is scattered and no one sees the same picture of each student. The director wants the single change that would MOST improve coordination.

Which action would MOST effectively improve the service’s information system?

A. Buy faster Wi‑Fi for tutors without changing data or process design
B. Build a more integrated process with shared data and clearer procedures across the team
C. Replace all tutors with automation as soon as possible
D. Focus only on making the logo and interface look more modern
Correct answer: B
Explanation
B is correct because the core problem is fragmented information and weak coordination across people and procedures. A shared process plus shared data improves the actual information system rather than only upgrading one technical component. A is wrong because faster Wi‑Fi does not solve inconsistent information flow by itself. C is extreme and ignores the value of people in the system. D focuses on appearance instead of underlying process and data quality.
Question 5
Application / evaluation
A retail manager is reviewing two possible technology projects. One project would add flashy new screens in stores, while the other would improve inventory accuracy, employee workflows, and the speed of restocking decisions. She wants the project that better reflects MIS thinking from the start of the course.

Which project should the manager choose if she wants the stronger information system improvement?

A. The screen project, because visible technology is always the clearest source of value
B. The inventory and workflow project, because it improves information quality and business process performance
C. The screen project, because hardware matters more than procedures and people
D. Neither project, because MIS is only about programming
Correct answer: B
Explanation
B is correct because Chapter 1 pushes managers to think about business value, process, and better decisions rather than just flashy tools. Improving inventory accuracy and workflows strengthens the system’s data and procedures, which can directly improve performance. A and C are wrong because they overvalue visible technology while ignoring process. D is wrong because MIS is broader than programming and includes management, decision-making, and business systems.

Vocabulary for Chapter 1

Specific definitions tied to the chapter slides and the kind of business-first framing your professor seems to like.

Information System (IS)

Core idea

An information system is a group of components that work together to produce information useful for a business or organization. In this chapter, the key idea is that an IS is not just technology. It also includes data, people, and procedures.

Citations: [S1], [S2], [W1]

Information Technology (IT)

Subset

Information technology usually refers to the hardware and software used in a system. It matters because many students confuse IT with IS, but the course keeps emphasizing that technology is only part of the larger system.

Citations: [S2], [W2]

Five-Component Model

Framework

The five-component model says an information system includes hardware, software, data, procedures, and people. Managers use this model to diagnose why a system is working poorly and to avoid assuming that software alone can solve every problem.

Citations: [S1], [S3]

Procedure

Process

A procedure is the method or set of steps people follow when using an information system. Procedures matter because even strong software can fail if people do not know what steps to follow or if the process itself is poorly designed.

Citations: [S3], [W3]

Data

Resource

Data are recorded facts or observations. In MIS, the point is not just that data exist, but that clean and shared data allow an organization to create better information and make better decisions.

Citations: [S3], [W4]

Digital Transformation

Business change

Digital transformation is the use of digital tools to redesign or improve how a business creates value and performs work. The important idea is that it changes processes and organizational behavior, not just the visible technology.

Citations: [S4], [W5]

Collaboration

People component

Collaboration is coordinated work across people and roles to achieve a shared result. In the opening chapter, collaboration matters because information systems often fail when people cannot share data, coordinate actions, or understand common procedures.

Citations: [S5], [S6]

Business Process

Operations

A business process is a set of related activities that accomplish a business objective. MIS focuses on these processes because information systems create value when they improve how work is actually performed, not when they only add flashy technology.

Citations: [S4], [S6], [W6]
